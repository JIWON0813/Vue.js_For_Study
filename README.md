# Vue.js_For_Study

<h1>Vue.js 공부 및 실습을 위한 레퍼지토리(Do it! Vue.js 입문 책 활용)</h1>

<h3>1일차</h3>
1.뷰 인스턴스의 유효범위(index.html 주석 참조)

뷰 라이브러리 파일 로딩 -> 인스턴스 객체 생성(옵션 속성 포함) -> 특정 화면요소에 인스턴스를 붙임  
-> 인스턴스 내용이 화면 요소로 변환 -> 변환된 화면 요소를 사용자가 최종 확인
